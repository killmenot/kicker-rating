# Dashboard application for kicker rating

## Dependencies

* NodeJS 0.10.33+
* PostgreSQL 9.3.5

## Setup
* `git clone git@github.com:killmenot/kicker-rating.git`
* `cd kicker-rating && make install`
* `make init-database`

## Migration
* `make migrate`

## Run
* `redis-server`
* `make run`
